# Intelligenza Artificiale
 studio dei fondamenti teorici e delle tecniche algoritmiche per la creazione di sistemi intelligenti.
- Ricerca nello spazio degli stati (A*, BFS, DFS)
- Giochi e avversari (Minimax, Alpha-Beta Pruning)
- Rappresentazione della conoscenza e Ragionamento
- Introduzione al Machine Learning e Reti Neurali
